# Crafting Interpreters (Rust)

A Rust implementation of the Lox interpreter from [Crafting Interpreters](https://craftinginterpreters.com/) by Robert Nystrom.

## Building

```bash
cargo build
```

## Running

**REPL** (interactive prompt):

```bash
cargo run
```

**Run a script file:**

```bash
cargo run path/to/script.lox
```
